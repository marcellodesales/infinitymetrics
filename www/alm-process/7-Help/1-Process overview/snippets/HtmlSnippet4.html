CollabNet's baseline process comes with a set of ready-to-use assets:

  • Project pages. A project owner can use these to manage project information, share documentation, provide views into project artifacts, and much more.

  • Project Tracker artifacts. Requirements, defect reports, and action items help plan and track the scope of releases and iterations through the lifecycle of the project.

  • Project Tracker queries. Project members can quickly query artifacts to get started on their own assignments, or to get a deep dive into project scope.

  • Document templates. A starting point to create project documentation.
