Some left navigation menu were just renamed and others included to better organize the project information. Here is the description of each of the menu items, highlighting the ones that are new, grouped into categories:

Project Communication

Announcements View the announcements (RSS'able project headlines) currently active on this project, or create new announcements for your project.
Mailing lists View the mailing lists currently available in this project, browse/search mailing list archives, or create new mailing lists for your project.
Discussion forums View the discussion forums currently active on this project, browse/search discussion forum archives, or create new forums for your project.
Meeting Channels
 NEW ITEM: Where the team gets together to develop PPM. It includes the list of communication tools used by the team such as IRC, Skype contact list and DimDim group presentation room.

Project Orchestration

Development Process NEW ITEM: Description of the Agile/Scrum development process used by the team members, specially important for those members with Developer role.
Milestones NEW ITEM: All the information about the milestones for the project. It must include the list of all final artifacts produced by each of the milestones. It should also reflect the releases from the Agile process, but the term will be interchangeable.
Team Calendar NEW ITEM: The team calendar with the activities defined during the team meetings.
Project Tracker RENAMED ITEM: This item continues to be ISSUE TRACKER. It was renamed to reflect that we have much more than issues to track. Enter, query, view, and edit artifacts defined within this project (for example: requirements, defect reports, and tasks).
Project Wiki
  NEW ITEM: The team's wiki page for continuous collaboration with documentation.

Project Artifacts

Developer Environment
 NEW ITEM: Description of the development environment used, configuration, and optional virtualization images for the common integrated development environment.
Subversion Repository
 RENAMED ITEM: Browse the Subversion repository in this project, download individual files, and see color-coded "diffs" for specific versioned files.
Documents & files Shared documents and files in the final version for deliverables.
Developer Corner NEW ITEM: Useful resources for developers such as design-patterns, code snippets, documentation, tools, etc.
