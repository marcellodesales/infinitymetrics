    Vision

    For Academia and the Industry, who require automated or semi-automated tools to collect metrics for teamwork assessment in collaborative software development environments such as Java.net, Metrics (Infinity Metrics) will be a web-based software that provides an online metric analysis interface to support project owners and managers. Unlike the prototype previously developed by Statwidgets, which neither properly parses XML documents nor uses a database for persisting the extracted data, PPM-8 will efficiently collect metrics data from any Java.net public project and store them in a MySQL database to be later analyzed in web browsers.

    Background Information

    PPM-8 will be based on research findings from professors and professionals who identified the types of metrics that are useful for team assessment in collaborative software development. The information is described in the PPM project. It will be developed in PHP5, MySQL5 and use the most exciting AJAX interfaces.
