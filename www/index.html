If you were not added before the inception of the project, please contact the owner at owner@ppm-8.dev.java.net. Identify yourself and why you have requested a role. Don't forget to include your name, username, and email address.

Participation:

If you are a member, you can see the current list of members here

If you are not a member, you can request membership to this project here.


ATTENTION: All project members with Developer role MUST be members of the following list:

 

  • commits: follow up with code checked in into the repository. It will alert you of code you possibly depend on;
  • dev: it will be used to discuss regular/daily development situations;
  • issues: important to all developers since the distributed development is managed by issue tracking;
  • users: possibly the main communication channel with users outside the project development scope. Our professors/clients will potentially be sending emails to this list.
Review the mailing lists, use email filters/tags/folders to better filter your messages.
