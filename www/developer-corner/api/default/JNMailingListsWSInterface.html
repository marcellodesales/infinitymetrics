Interface JNMailingListsWSInterface

Description
Description | Methods (details)

Retrieves the list of

  1. mailing lists = https://ppm-8.dev.java.net/servlets/ProjectMailingListList - This list is not updated that often.

2. discussion forums = https://ppm-8.dev.java.net/servlets/ProjectForumView

  • This forum list tend to grow faster than the mailing list.
This interface will provide separate lists or a complete set.

If the mailing list is dev@ppm-8.dev.java.net, the service must only retrieve "dev".


Project owners has the complete visibility, while regular users just the public ones. They are captured from the  website https://ppm-8.dev.java.net/servlets/ProjectMailingListList
Fromt the website, there's no way to determine if a mailing list if public or private. There's a need to login  twice to the service.
