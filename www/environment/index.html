PPM-8 Virtualized Environment

In order to maintain a single image of the development environment, the team will optionally be using a VirtualBox image with the entire installation of the necessary software to develop. VirtualBox is a free virtualization tool provided by Sun to the community! More information go to http://www.virtualbox.org/. The PPM-8 virtualization image includes the entire infrastructure discussed on the PPM project during the Milestone 0.
  • Windows XP SP3;
  • Apache 2.2, PHP 5.2.4, xdebug 2.0.3, MySQL 5, Netbeans 6.5;
  • CollabNet Subversion 1.5, Tortoise Subversion Plug-in;
  • Additional software for development such as TOAD (MySQL designer) and others.
In real-world projects, virtualization became a de-facto standard in customizing development builds for the entire team. In fact, developers only work in virtualized machines instead of their own when it comes to bigger enterprises. As a consequence, every developer will be working with the same environment profile for development activities and will be able to make easier references to their peers accordingly. Moreover, operations technicians will promptly make one single update to the environment and every one with the same virtual machine profile will be with their development environment updated.

For our PPM-8 development environment, we will be updating the profile once we have a big change with the softwares necessary to develop out application.
